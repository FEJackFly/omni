# Svelte Omni 地图平台 —— 需求规划

---

## 1. 地图基础建设

### 1.1 主题配置

- **地图样式集**
  - 白色主题
  - 黑色主题
  - 蓝色主题
  - 卫星/地形图
- **视觉美化**
  - 建筑物 3D 效果优化
  - 道路宽度与圆角美化
- **性能优化**
  - 图层 `minzoom` 调整
  - 移除冗余图层

### 1.2 天空插件

- 根据不同地图主题，动态切换对应的天空（Sky）配置

### 1.3 画图工具

- 支持点、线、面（Polygon）的绘制
- 支持导出 GeoJSON 数据
- 测距功能
- 测面积功能

---

## 2. 图层管理

### 2.1 MapLibre 图层

| 图层类型         | 说明                       |
| ---------------- | -------------------------- |
| OSM 底图         | OpenStreetMap 标准底图     |
| GeoJSON 矢量图层 | 自定义矢量数据叠加         |
| 辅助图层         | 测距、测面积的临时绘制图层 |

### 2.2 Deck.gl 图层（自动驾驶渲染）

| 图层类型       | 说明               |
| -------------- | ------------------ |
| 车体坐标系渲染 | 显示车辆自身坐标轴 |
| 轨迹线渲染     | 车辆历史/规划轨迹  |
| 障碍物渲染     | 检测到的目标物体   |
| 雷达点云渲染   | LiDAR 点云可视化   |

### 2.3 Three.js / Babylon.js 图层

- 特效图层（粒子、光效等高级 3D 渲染）

---

## 3. 视图管理

### 3.1 标准视图

- **2D 鸟瞰视图**
- **3D 透视视图**

### 3.2 自动驾驶专属视角

- **车辆跟随模式**：视角始终跟随车辆移动并保持朝向
- **车辆固定模式**：视角锁定在车辆正上方，不旋转
- **自由拖拽模式**：用户手动拖拽后，视角保持在当前位置继续跟随车辆平移（不强制拉回）

---

## 4. 数据模拟

### 4.1 动态车辆位姿模拟

基于输入的 GeoJSON 路线，生成仿真车辆运动数据，可配置参数：

- 车速设置
- 车辆数量设置
- 输出字段：位置坐标（x, y）+ 航向角（Heading）

### 4.2 单车场景模拟

- 单车障碍物模拟
- 单车 50 米范围轨迹模拟
- 单车雷达点云模拟
